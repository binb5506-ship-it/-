# 蓝桥杯单片机资料整理

这个仓库整理了蓝桥杯单片机方向的 C51 / STC15F2K60S2 学习资料、练习工程和历届省赛参考工程。工程主要使用 Keil uVision 打开，代码围绕数码管、按键、LED、IIC、DS18B20、DS1302、DAC 等蓝桥杯常见外设展开。

## 内容概览

- Keil C51 工程：38 个
- `main.c` 主程序文件：38 个
- 已生成的 `.hex` 文件：35 个
- 主要芯片平台：STC15F2K60S2 / 8051 C51
- 常见工程结构：`User/main.c`、`Driver/*.c`、`Driver/*.h`、`*.uvproj`

## 开发环境

建议使用下面的环境打开和编译：

- Keil uVision 5
- C51 编译器
- STC15F2K60S2 芯片支持包
- STC-ISP 下载工具
- 蓝桥杯单片机竞赛开发板

如果 Keil 打开工程时报 `main.c was not found`，不要直接从压缩包或临时目录里打开 `.uvproj`。应先完整解压整个资料目录，再从解压后的固定目录打开工程文件。

## 如何打开工程

以第十二届省赛第一次工程为例：

```text
第十二届省赛（第一次）/User/1.uvproj
```

打开方式：

1. 启动 Keil uVision。
2. 选择 `Project -> Open Project`。
3. 打开对应目录下的 `.uvproj` 文件。
4. 确认工程中的 `User/main.c` 和 `Driver/*.c` 均能正常展开。
5. 点击 Build 编译，生成 `.hex` 后使用 STC-ISP 下载到开发板。

## 常见外设模块

资料中出现较多的驱动模块包括：

- `led.c / led.h`：LED 显示控制
- `seg.c / seg.h`：数码管动态扫描显示
- `key.c / key.h`：独立按键、矩阵按键扫描
- `iic.c / iic.h`：IIC 总线及 EEPROM / DAC 相关操作
- `onewire.c / onewire.h`：DS18B20 单总线温度采集
- `ds1302.c / ds1302.h`：DS1302 实时时钟
- `init.c / init.h`：系统初始化、外设初始化

## 学习建议

建议按下面顺序学习：

1. `初始化`、`点灯`：熟悉 IO 控制和基础工程结构。
2. `中断`、`中断（含定时器中断）`：理解定时器、系统节拍和中断服务函数。
3. `数码管动态显示`：掌握数码管位选、段选和动态刷新。
4. `矩阵按键与独立按键`：掌握按键扫描、消抖和边沿触发。
5. `ds1302时钟实时显示`、`底层练习`：练习常见通信外设。
6. 历届省赛工程：重点看任务拆分、显示界面切换、参数设置、状态机和调度器写法。

## 代码风格说明

部分工程采用简易任务调度器结构：

```c
typedef struct
{
    void (*task_func)(void);
    unsigned long int rate_ms;
    unsigned long int last_ms;
} task_t;
```

这类写法通常配合 1ms 定时器中断使用。中断负责维护 `uwTick` 系统时基，主循环中的调度器按照不同周期调用按键、显示、LED、温度采集、DAC 输出等任务。它不是 RTOS，而是单片机比赛中常用的协作式轮询调度。

## 注意事项

- 不建议直接在压缩包内部打开 Keil 工程。
- 不建议从 `AppData/Local/Temp` 等临时解压目录打开工程。
- 如果移动工程目录，应保持 `User`、`Driver`、`Objects` 等目录的相对位置不变。
- Keil 工程路径中包含中文通常可以使用，但如果遇到编译器路径异常，可复制到纯英文目录再试。
- `.hex` 是编译输出文件，源码学习应优先查看 `User/main.c` 和 `Driver` 目录。
